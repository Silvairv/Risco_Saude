# Risco_Saude

Análise de Fatores de Risco para a Saúde

Este projeto realiza uma análise exploratória de dados (EDA) em um conjunto de dados sintético focado em estilo de vida e previsão de risco à saúde. O objetivo é identificar padrões, correlações e a influência de diversas variáveis de estilo de vida em diferentes níveis de risco à saúde (baixo, médio, alto).

Principais Tópicos Abordados:

Preparação e Limpeza de Dados: Carregamento do dataset, verificação de valores ausentes e renomeação de colunas para maior clareza.

Análise de Correlações Numéricas: Exploração das relações entre variáveis como idade, peso, altura, sono e Índice de Massa Corporal (IMC).

Visualização de Dados Categóricos: Análise da distribuição do risco de saúde em relação a fatores como consumo de açúcar, hábito de fumar e consumo de álcool, utilizando tabelas de contingência e gráficos de barras.

Relação entre Idade e Risco de Saúde: Visualização da distribuição da idade para cada categoria de risco à saúde, buscando insights sobre como a idade pode influenciar o risco.

Tecnologias Utilizadas:
Python

Pandas: Para manipulação e análise de dados.

Plotly Express: Para criação de visualizações interativas.

Este projeto fornece uma base para entender os fatores que contribuem para o risco à saúde e pode ser expandido para a construção de modelos preditivos.
